# Data Analytics Pipeline using Apache Druid (ELT Architecture)

## 🚀 Overview
This project demonstrates an end-to-end data analytics pipeline using Apache Druid, BigQuery, and ELT architecture.

## 🏗️ Architecture
1. CSV dataset stored locally
2. Python script sends ingestion request to Apache Druid
3. Druid creates indexing tasks and ingests data
4. Data is transformed using Druid SQL (ELT approach)
5. Analytical queries are executed
6. Data visualized using Apache Superset and Tableau

## ⚙️ Tech Stack
- Apache Druid
- Python (requests)
- Google BigQuery
- dbt
- PostgreSQL (Neon)
- Apache Superset
- Docker

## 🔄 Key Features
- Automated data ingestion using Python and Druid API
- ELT-based transformation using Druid SQL
- Aggregation queries (revenue, energy consumption, trends)
- Creation of summary tables for optimization
- Dashboard visualization using Superset

## 📊 Example Queries
- Total Energy Consumption
- Total Revenue Generated
- Daily Revenue Trends
- Payment Method Distribution
